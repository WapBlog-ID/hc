<h1>Pertanyaan yang sering ditanyakan</h1>

<ul>

<li>

<h4>Q: Bagaimana membuat blog?</h4>

<h3>Melalui Panel MY</h3>

<ul>

<li>Buka Panel <a data-url="https://my.wapblog.id/panel/blog" class="switch-url">Kelola Blog</a></li>

<li>Klik <b>Add Blog</b></li>

<li>Masukan url blog yang kamu inginkan di kolom <u>Subdomain</u>. (contoh: <b>blogsaya</b>).</li>

<li>Beralih ke kolom <u>Domain</u>, pilih satu domain yang tersedia (contoh: <b>.bloger.id</b>)</li>

<li>Beralih ke kolom <u>Blog Name</u>, masukan nama blog yang akan menjadi judul utama blog kamu. (contoh: <b>Blog Saya</b>)</li>

<li>Beralih ke kolom <u>Blog Description</u>, masukan deskripsi yang menggambarkan aliran dan jenis blogmu. Masukan deskripsi yang singkat jelas dan mewakili semua tentang blogmu. (contoh: <b>Catatan harian saya</b>)</li>

<li>Klik tombol <b>add</b> dan kamu akan melihat blogmu sudah berhasil dibuat. Dan kamu bisa langsung melihat blogmu url <b>https://blogsaya.bloger.id</b></li>

</ul>

<h3>Melalui Panel Dash</h3>

<ul>

<li>Buka Dashboard <a data-url="https://dash.wapblog.id/panel/blog" class="switch-url">Blog</a></li>

<li>Klik tombol <b><i class="fa fa-plus"></i> Tambah</b> di pojok kanan atas</li>

<li>Masukan url blog yang kamu inginkan di kolom <u>Url Blog</u>. (contoh: <b>blogsaya</b>).</li>

<li>Beralih ke kolom <u>-- Domain --</u>, pilih satu domain yang tersedia (contoh: <b>.bloger.id</b>)</li>

<li>Beralih ke kolom <u>Nama Blog</u>, masukan nama blog yang akan menjadi judul utama blog kamu. (contoh: <b>Blog Saya</b>)</li>

<li>Beralih ke kolom <u>Deskripsi Blog</u>, masukan deskripsi yang menggambarkan aliran dan jenis blogmu. Masukan deskripsi yang singkat jelas dan mewakili semua tentang blogmu. (contoh: <b>Catatan harian saya</b>)</li>

<li>Lanjut ke kolom <u>--Pilih Jenis Blog--</u>, dan pilih satu yang menurutmu cocok dengan kategori blogmu</li>

<li>Klik tombol <b>Tambah</b> dan kamu akan melihat blogmu sudah berhasil dibuat. Dan kamu bisa langsung melihat blogmu url <b>https://blogsaya.bloger.id</b></li>

</ul>

</li>

<li>

<h4>Q: Bagaimana menulis posting?</h4>

<h3>Melalui Panel MY</h3>

<ul>

<li>Buka <a data-url="https://my.wapblog.id/panel/post?act=add" class="switch-url">Add Post</a> melalui <u>menu</u>. Jika kamu belum login blog, kamu akan diarahkan ke halaman blog dan silahkan pilih blog yang ingin kamu tambah posting</li>

<li>Setelah terbuka halaman menulis posting, kamu bisa mengisi judul pada kolom <u>Title</u> minimal 5 karakter</li>

<li>Tuliskan konten posting mu pada kolom <u>Content</u>. <b>Perlu diingat</b>, jika kamu menulis melalui veri <b>MY</b> kamu harus menuliskan dalam bentuk html penuh.</li>

<li>Pilih kategori posting pada kolom <u>Category</u> (Tampil hanya ketika kamu memiliki kategori).</li>

<li>Masukan tags pada kolom <u>Tags</u> (ini adalah meta data yang mirip dengan kategori).</li>

<li>Masukan tanggal posting dengan format <u>TTTT-BB-DD JJ:mm</u> (contoh: 2021-01-26 19:31 untuk membuat posting terbit pada 26 Januari tahun 2021 pukul 19:31)</li>

<li>Tekan tombol <u>Publish</u> untuk menerbitkan posting, atau <u>Draft</u> untuk menyimpan dipanel.</li>

</ul>

</li>

<li>

<h4>Q: Apa itu tema ponsel?</h4>

<p>Tema ponsel adalah tema yang dibuat untuk mendeteksi peramban ponsel berteknologi J2ME(Lihat <a class="switch-url" data-url="https://en.wikipedia.org/wiki/Java_Platform,_Micro_Edition">Wikipedia</a>) atau ponsel lainnya yang tidak mendukung teknologi WebKit (Lihat <a class="switch-url" data-url="https://id.wikipedia.org/wiki/WebKit">Wikipedia</a>).</p>

</li>

<li>

<h4>Q: Apa itu tema ponsel High-End?</h4>

<p>Pada dasarnya, WapBlog.ID memberikan kamu 3 mode tampilan yaitu ponsel, high end, dan desktop. High end adalah mode tema yang akan ditampilkan ketika kamu mengunjungi blog dengan peramban yang memiliki teknologi WebKit (Lihat <a class="switch-url" data-url="https://id.wikipedia.org/wiki/WebKit">Wikipedia</a>). Tema ini secara default mati dan kamu harus menyalakan pada <b>pengaturan dasar blog</b> kolom <b>Tema Ponsel High-End</b>. Jika pengaturan ini mati, maka otomatis peramban yang memiliki teknologi Webkit akan mengakses tema Desktop yang mana tema default yang kami sediakan mendukung tampilan responsive.</p>

</li>

<li>

<h4>Q: Saya sudah merubah CSS ponsel, mengapa tidak berubah?</h4>

<p>Secara default semua CSS pada blog tidak diatur untuk proses cache atau penyimpanan ke lokal memori peramban. Namun, beberapa peramban tetap <b>memaksa</b> akan menyimpan CSS ke memori peramban (Mode hemat data) pada perangkat yang mana hal ini bisa menambah kecepatan ketika kamu berseluncur disemua situs atau blog yang kamu kunjungi termasuk diluar WapBlog.ID.</p>

<p>Ada beberapa cara untuk bisa langsung melewati proses caching peramban:</p>

<ul>

<li>Menghapus cache untuk situs atau blog yang spesifik dipengaturan peramban.</li>

<li>Menggunakan bypass dengan menambahkan Search Query (misalnya <code>?v=1</code> untuk pengeditan pertama)</li>

<li>Menggunakan mode Incognito atau mode Samaran pada peramban. (Tentang Incognito: <a class="switch-url" data-url="Chrome https://support.google.com/chrome/answer/95464?hl=id">baca disini</a>, Firefox <a class="switch-url" data-url="https://support.mozilla.org/id/kb/Penjelajahan%20Pribadi">baca disini</a>)</li>

<li>Paksa Reload dengan <code>CTRL+Shift+R</code> pada OS Windows atau <code>Command+Shift+R</code> untuk OS Mac di peramban desktop.</li>

</ul>

</li>

</ul>
